## 💡 UniApp 端定位

**UniApp 端 = 移动端轻量化管理工具**

- ✅ 适用场景：日常查看、简单操作、移动办公
- ❌ 不适用：复杂数据分析、批量操作、系统配置
- 🌐 复杂功能：请使用 Web 管理端

## 🎯 开发优先级建议

### P0 - 核心缺口（影响业务闭环）

1. **教练端工作台**（0%）
   - 工作台、我的课程、课表、预约管理
   - 影响教练日常工作流程

2. **场馆管理者端业务功能**（33%）
   - 场地管理、器材管理、穿线服务
   - 影响场馆日常运营

### P1 - 重要增强（提升体验）

1. **用户端数据真实化**
   - 首页余额、会员等级、预约提醒
   - 提升用户体验

2. **场馆管理者端业务功能扩展**
   - 根据实际需求逐步补充
   - 优先级低于教练端

### P2 - 可选优化（非必需）

> 💡 以下功能建议在 Web 管理端完成，UniApp 端非必需

1. 独立工作台页面（各端）
2. 复杂数据分析和报表
3. 批量操作功能
4. 系统配置管理

## 📝 文档使用说明

### 查看某个角色的页面清单

点击上方文档索引中的链接，查看对应角色的详细页面清单。

### 文档结构说明

每个角色文档包含以下章节：

1. **工程现状** - 当前已实现的页面列表
2. **页面完成度统计** - 量化的完成度数据
3. **Web 端对照** - 与 Web 管理端的功能对照
4. **待补充页面** - 尚未实现的功能清单（带勾选框）
5. **实施注意事项** - 开发时的技术要点
6. **总结** - 当前状态和优先级建议

### 更新文档

- 完成新页面后，在对应文档中勾选 `- [x]`
- 更新完成度统计表
- 更新文档底部的"最后更新"日期
