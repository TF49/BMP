## 1. 项目概述

### 1.1 项目简介

羽毛球管理系统（Badminton Management Platform，简称BMP）是一个基于 Spring Boot + Vue 3 开发的现代化前后端分离应用，用于高效管理羽毛球场馆、会员、场地预约、教练课程、赛事组织等相关业务。

### 1.3 快速启动

```bash
# 1. 数据库初始化
mysql -u root -p
CREATE DATABASE badminton CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
mysql -u root -p badminton < src/main/resources/sql/badminton.sql

# 2. 修改配置
# 编辑 src/main/resources/application.properties
# 建议使用环境变量（避免在配置文件写明文）
$env:DB_USERNAME="root"
$env:DB_PASSWORD="你的数据库密码"
$env:JWT_SECRET="请替换为至少32字节的高强度密钥"

# 3. 启动后端 (端口: 9090)
mvn spring-boot:run

# 4. 启动前端 (端口: 8080)
cd vue && npm install && npm run serve
```

**默认账号**: `admin` / `admin123` (角色: PRESIDENT)

### 4.3 权限实现

**三层权限防护**:
1. **Controller层**: `@PreAuthorize` 注解进行方法级权限控制
2. **Service层**: 数据范围校验，确保数据级隔离
3. **前端层**: 路由守卫 + 按钮级权限指令

**实现示例**:
```java
// 后端 - Controller层
@PreAuthorize("hasAnyRole('PRESIDENT', 'VENUE_MANAGER')")
public Result<Object> update(@RequestBody Entity entity) { ... }

// 后端 - Service层
if (SecurityUtils.isVenueManager()) {
    Long vId = SecurityUtils.getCurrentUserVenueId();
    if (!vId.equals(entity.getVenueId())) {
        throw new RuntimeException("权限不足：只能操作自己场馆的数据");
    }
}
```

```vue
<!-- 前端 - 按钮级权限 -->
<el-button v-permission="['PRESIDENT']">删除</el-button>
<el-button v-permission="['PRESIDENT', 'VENUE_MANAGER']">编辑</el-button>
```

## 6. 开发规范

### 6.1 代码规范

- **后端**: 遵循阿里巴巴Java开发手册，使用Lombok简化代码
- **前端**: 遵循Vue.js风格指南，组件命名使用PascalCase

### 6.2 Git提交规范

```
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式调整
refactor: 代码重构
perf: 性能优化
test: 测试相关
chore: 构建过程或辅助工具的变动
```

### 6.3 数据库规范

- **表名**: 小写 + 下划线 (`sys_user`, `biz_booking`)
- **主键**: 统一使用 `id`
- **时间字段**: `create_time`, `update_time`
- **逻辑删除**: `del_flag` (0-正常, 1-删除)

### 6.4 API规范

- **统一返回格式**: `Result<T>` (code, message, data)
- **RESTful风格**: GET查询、POST新增、PUT更新、DELETE删除
- **分页参数**: `page`（默认1）、`size`（默认10，最大100）
- **认证方式**: JWT Token（请求头：`Authorization: Bearer {token}`）

**详细API文档**: 参考 `API文档.md`
