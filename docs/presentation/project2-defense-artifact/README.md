# 项目二答辩 · React 工程

基于 [网页制品构建器](https://github.com/) 流程：Vite + React + Tailwind + react-bits 风格组件。

## 命令

```bash
pnpm dev      # 本地调试（http://localhost:5173）
pnpm build    # 生产构建（dist/）
pnpm bundle   # 单文件 bundle.html（内联 JS/CSS，可覆盖 ../project2-defense.html）
```

蓝图数据来自同级目录 [`../project2-blueprint.json`](../project2-blueprint.json)。
